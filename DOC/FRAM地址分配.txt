+----------------+--------------------+--------------------------------------------------------+
+   地址范围      +     存储内容       +                       说    明                         +
+----------------+--------------------+--------------------------------------------------------+
+ 0x0000-0x0001  + 初始化标志：0x55AA  + 第一次上电时写入，之后上电读取判断                       +
+----------------+--------------------+--------------------------------------------------------+
+ 0x0002-0x00E6  +    ATT校准参数      + 0x02-0x70为ATT1,0x78-0xE6为ATT2                        +
+----------------+--------------------+--------------------------------------------------------+
+ 0x0100-0x0140  +     实时电能        + A、B母线分别有4个有功电能、4个无功电能                   +
+----------------+--------------------+--------------------------------------------------------+
+ 0x0200-0x02AD  +     设置参数        + 系统设置+支路开关设置                                   +
+----------------+--------------------+--------------------------------------------------------+
+ 0x02B0-0x02B6  +    IAP相关标志      + 1个初始化标志、1个IAP复位标志、1个下载完成标志           +
+----------------+--------------------+--------------------------------------------------------+
+ 0x0300-0x1497  +     历史告警        + 1个>500条标志、1个条数索引、告警记录（0x09*500byte）     +
+----------------+--------------------+--------------------------------------------------------+
+ 0x1500-0x2405  +     历史电能        + 初始化标志、时基、保存标志，历史电能（32*10*12Byte）     +
+----------------+--------------------+--------------------------------------------------------+
